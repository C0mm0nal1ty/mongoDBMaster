# Project overview

We will be making an application that uses a MongoDB, mongoose, and Mocha as a testing environment.
Overview of operations:
Mocha starts, Tell mongoose to connect to Mongo, wait, Mongoose connects to Mongo,
Connection successful? Run tests, Connection failed? show error.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.
